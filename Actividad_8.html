Adicionalmente para poder instalar y configurar el Active Directory es recomendable seguir los pasos del siguiente tutorial

Active Directory y algunos ejemplos de sus usos:

El Active Directory es un servicio de directorios que funciona para administrar dominios y conjuntos de procesos y servicios. Un servidor que corre en el Active Directory Domain Service es un contolador de dominio. Autentica y autoriza a todos los usuarios en una red de tipo dominio como lo veremos en la siguiente actividad.

El AD DS verifica las credenciales de los usuarios y define los derechos de acceso que estos tienen. Tambien puede utilizar Servicios de Certificados en su AD CS (Active Directory Certificate Services) con los cuales puede hacer uso de infraestructuras de llave publica (como PGP, GPG etc.) para validar llaves y permitir accesos. Los archivos pueden ser encriptados con estas llaves y se pueden mandar correos (o establecer un servidor de correos encriptados seguros como tal) mediante esta infraestructura.
