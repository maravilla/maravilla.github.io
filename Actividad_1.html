Cómo instalar Linux como sistema operativo nativo

El sistema operativo a instalar será Manjaro, una versión de linux basada en la distribución de ArchLinux conocido por ser rápido, poderoso, ligero y siempre actualizado. Manjaro agrega una capa adicional de pruebas para brindar una versión estable para los usuarios y agrega una interfaz de usuario agradable para usuarios no avanzados en línea de comandos.

 

Características Únicas:

 

  • Gratis
  • Centro de control de descarga de Software Gratis (AUR)
  • Rolling Release asegurando la versión más actualizada sin tener que reinstalar
  • Protección de datos personales
  • Actualiza tu equipo cuando quieras
  • Extremadamente configurable
  • Fácil de cambiar entre Kernels
  • Dependiendo de la distribución requiere más conocimiento previo
  • Software proprietario límitado

Se necesitarán:

  • Una memoria USB con al menos 4GB de capacidad total.
  • Conexión a internet
  • Una máquina adicional para generar medio de instalación

Requisitos de instalación para Manjaro de 64 bits:

  • 1 gigabyte (GB) de memoria RAM
  • 30 gigabytes (GB) de espacio en disco duro
  • Procesador de 1 gigahertz (Ghz) 
  • Tarjeta de salida de video HD y monitor compatible
  • Conexión a internet durante la instalación

Para comenzar la instalación hay que dirigirse a la página de descargas de manjaro.org.

En esta ocasión se descargará la versión XFCE de Manjaro.

 

XFCE y las demás opciones disponibles son entornos de escritorio con sus respectivos administradores de ventanas que te distintas versiones de entornos de escritorios para utilizar. Se usará XFCE gracias a su familiaridad con otros sistemas y su tamaño reducido de descarga.

También se tendrá que descargar la herramienta rufus que nos servirá para crear nuestro medio de instalación.

Una vez terminadas ambas descargas hay que correr rufus con permisos de Administrador y seleccionar los siguientes parámetros:

  • Device: Nuestra USB
  • Boot selection: Disk or ISO image y seleccionar el .iso que se descargó
  • Partition Scheme: MBR partition scheme for BIOS or UEFI"
  • START

Los datos de la USB serán destruídos durante este proceso, asegurarse de que se hizo un respaldo y oprimir continuar.

Una vez terminado este proceso podemos insertar esta USB en la máquina destino y bootear el sistema a través de ella. Esto consiste en oprimir cierta tecla mientras la pantalla de booteo de la computadora aparece al prenderla, y es dependiente de los BIOS o versión de UEFI de cada sistema. Una vez encontrada la manera de bootear por usb comienza la instalación de nuestro sistema.

 

Al ser bienvenido por la interfaz gráfica iniciamos la instalación.

Se selecciona el lenguaje a ser usado para el sistema y durante la instalación.

Se selecciona la zona horaria en la que será utilizado el equipo.

También se selecciona el formato del teclado utilizado y el lenguaje.

En nuestro caso estamos asumiendo que el sistema será el único sistema operativo en el disco duro. Existe la posibilidad de que el instalador detecte otros sistemas operativos, lo cual despliega la opción de instalar un sistema dual conservando ambos sistemas.

La partición que estamos utilizando es un encogimiento de un disco duro existente y hemos booteado directo a ella, en caso de tener otro sistema se puede reemplazar y borrar todo. Seleccionamos esta opción (instalar propio bootloader).

Se crea un nombre de usuario y una contraseña para el uso del equipo y de tareas de administrador, y se comienza la instalación.

Al terminar podemos reiniciar y bootear a la partición o al bootloader del equipo dependiendo de los sistemas instalados.

 

 Cómo instalar Windows 10 como sistema nativo

 

El sistema a instalar será Windows 10 Home en su versión oficial. Windows tiene la ventaja de ser de los sistemas operativos más utilizados y con mayor compatibilidad con software proprietario en el mercado. 

 

Características Únicas:

 

  • Proprietario (hay que pagar)
  • Software adicional proprietario bajado de internet o comprado
  • Control reducido sobre datos personales recaudados
  • Protección de datos personales
  • Actualizaciones forzadas por defecto
  • Configuración limitada
  • Familiar y fácil de utilizar
  • No requiere mucho conocimiento previo para su uso
  • Servicios proprietarios como Cortana y Xbox

Se necesitarán:

  • Medio de instalación oficial o descarga de imagen para usb
  • Conexión a internet

Requisitos mínimos de instalación para Windows 10:

  • Procesador de 1 gigahertz (GHz) or mejor
  • 1 gigabyte (GB) de memoria RAM para versión de 32-bits or 2 GB para versión de 64-bits
  • 16 GB  de espacio en disco duro para 32-bits ó 20 GB para sistema de 64-bits
  • Tarjeta gráfica DirectX 9 o mejor con controlador WDDM 1.0
