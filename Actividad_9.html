Cómo crear usuarios y grupos de usuarios en Windows Server 2019

 

Una vez configurado el Active Directory correctamente (lo detallado en la actividad anterior sobre: IP estática, servidor DNS y DHCP y el upgrade a domain controller) podemos comenzar a agregar a nuestros usuarios al dominio de la siguiente manera:

 

Navegamos a Active Directory Users and Computers en nuestro Dashboard

 

Damos Click derecho en nuestro desgloce del servidor local en la carpeta de usuarios y damos clic en crear nuevo usuario

 

 

Llenamos el formato de nuevo usuario con el nomre y login del usuario

 

 

Creamos una nueva palabra de acceso

 

 

Y de esta manera podemos instalar los usuarios que queramos.

 

 

Para crear grupos repetimos el proceso de igual manera ahora seleccionando crear nuevo grupo

 

 

Llenamos el formato del Grupo

 

 

Una vez creado el grupo podemos dar click derecho en un usuario para agregarlo al grupo

 

 

Buscamos el nombre por grupo y lo verificamos y lo podemos agregar.

 

 

Podemos ver entonces los usuarios del grupo al inspeccionarlo

 

 

Podemos crear multiples grupos con distintas características y privilegios y con distintos usuarios.

 

 

 

Y de esta manera tenemos 4 grupos de usuarios diferentes para nuestros usuarios instalados.
