Cómo crear un juego multiplataforma con Python

Nuestro programa se va a basar en la biblioteca de Python Tkinter ya que es una interfaz a la interfaz gráfica de Tk. Tk y Tkinter se encuentran disponibles en la mayoría de las distribuciones de UNIX y en Windows, es por esto que la utilizaremos como base de nuestro programa gráfico en 2D

Nos vamos a basar en el programa de este repositorio https://github.com/maravilla/SnakeGame creado por el usuario surajsingbisht054

El programa fuente completo puede ser encontrado En este repositorio

Hay que destacar que la sentencia que nos permite ejecutar este programa en múltiples sistemas operativos (Unix y Windows) es el shebang #! que ponemos en la primer línea de nuestro código fuente:
#!/usr/bin/python
Ésta es la línea que nos va a permitir correr este código en UNIX ya que para crear el ejecutable esta predirectiva le indica al interprete la manera en la que puede transformar el código fuente a un ejecutable adecuado de manera automática Ésta predirectiva puede tener impacto en Windows cuando existen múltiples interpretes de Python instalados y deseamos seleccionar uno manualmente.
